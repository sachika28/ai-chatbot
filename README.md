# 🤖 My Personal AI Chatbot

A conversational AI chatbot built with Python, Google Gemini API 
and Streamlit — runs directly in your browser!

## Features
- 💬 Multi-turn conversation with memory
- 🧠 Powered by Google Gemini 2.5 Flash
- 🌐 Clean web interface via Streamlit
- 📅 Aware of today's date and context
- 🔒 API key secured using .env file

## Tools Used
- Python
- Google Generative AI (Gemini 2.5 Flash)
- Streamlit
- python-dotenv

## How to Run
1. Clone this repository
2. Install requirements:
   `pip install google-generativeai streamlit python-dotenv`
3. Create a `.env` file and add:
   `GEMINI_API_KEY=your-api-key-here`
4. Get your free API key at aistudio.google.com
5. Run: `streamlit run chatbot.py`
